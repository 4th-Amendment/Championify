What is LoL Masters?

LoL Masters is a tool which provides you useful statistics about all the champions of the league.

All the data is provided by constantly collecting the latest solo queue matches played by summoners currently placed in Diamond, Master or Challenger tier from EUW, NA and KR regions.

Every info you can find here is then updated every hour and reset whenever a new major League of Legends patch is released.
